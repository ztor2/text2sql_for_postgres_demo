# Text-to-SQL Generator for PostgreSQL(Demo)


- 자연어 질의를 SQL 문으로 변환할 수 있습니다.
- DB를 연결하여, DB의 스키마 정보를 반영한 SQL을 작성할 수 있습니다.
- Huggingface 기반 Private LLM 시스템을 적용하여 데이터 보안을 준수합니다.
